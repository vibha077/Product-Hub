# React + Redux Product Management Application

**Live Demo:**(https://product-hub-xexa.vercel.app/#/login))  

# Project Overview

This project is a React + Redux based Product Management Application developed following modern industry best practices.  

It demonstrates:

- Structured project architecture
- Redux-based global state management
- JWT-based authentication
- API integration
- CRUD operations
- Clean UI with proper error handling
- Secure token storage 
- React (Latest version with functional components & hooks)
- Redux Toolkit
- React Redux
- Redux Thunk (for async operations)
- React Router DOM
- Axios (for API calls)
- Local Storage (JWT storage)
- Git & GitHub
